# Games & Life · 游戏中心

小游戏合集主页，托管于 GitHub Pages：<https://hufaei.github.io/games/>

## 添加新游戏

编辑 `index.html` 顶部的 `GAMES` 数组，追加一项即可：

```js
{
  title: '游戏名',
  desc: '一句话介绍',
  icon: '◇',            // 卡片图标（任意字符/emoji）
  accent: '#ff7a3c',    // 卡片主题色
  tags: ['类型'],
  url: 'https://hufaei.github.io/<repo>/'
}
```
